# Leonid AI Lab

Инженерная лаборатория по подготовке AI Automation Engineer.

## Цель

Создать портфолио коммерческих AI-проектов и выйти на международный уровень разработки.

## Структура

- `docs/` — документация
- `modules/` — учебные модули
- `projects/` — практические проекты
- `portfolio/` — проекты для работодателей
- `prompts/` — библиотека промптов
- `snippets/` — полезные фрагменты кода
- `templates/` — шаблоны
- `archive/` — архив
